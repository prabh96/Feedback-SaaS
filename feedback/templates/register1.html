Meaningful Feedback


About


The meaningful feedback app is a web application for achieving feedback from different online learning websites on the usefulness of its content. The app can be embedded on the publisher's website with an auto generated iframe link. Once embedded a user can sign-up/log into the app and vote on the two options- Affective and Cognitive with their respective choices. Each user can vote only once after which the total number of votes recorded for the website are displayed.
